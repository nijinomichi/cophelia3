## Interpretation boundary

The protocol combines scientific notation, poetic language, and social implementation. These layers remain distinct:

- equations are reviewed as scientific claims;
- poems are interpreted as poems;
- project metrics require an explicit method before they are treated as measurements.

This separation is not a failure of synthesis. It is what prevents synthesis from becoming soup with a quantum garnish.
